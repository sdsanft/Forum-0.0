# Full Stack App
##### By Sam Sanft
My first attempt at creating a full stack web application. Still a work in progress.
### Current Technologies
* Node.js
* Express.js
* React.js

### Coming Soon
* Database
  * MongoDB Atlas
  * MySQL

### How To Run
1. Download and install node
2. ` git clone https://github.com/sdsanft/NodeTest-0.0.git `
3. ` npm install `
4. ` npm run build `
5. ` npm run start `
